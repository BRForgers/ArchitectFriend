Overview

What is "Architect's Friend"?

"Architect's Friend" is a Chisel-Like Mod, that adds Building Blocks of all types. As other mods made by us, it requires BRForgersCore.

How I Install it?

  • If not yet, Install Minecraft Forge;
  • If you have, Delete any old versions of the ArchitectFriend.jar file from your ".minecraft/mods" folder.
  • If not yet, Download and put the BRForgersCore.jar file into your ".minecraft/mods" folder.
  • Download and put the ArchitectFriend.jar file into your ".minecraft/mods" folder.
  • Restart/Open Minecraft

Adding compatibility with Forge Microblocks:

  • Download ForgeMultipart and install it
  • Run Minecraft
  • Open the folder ".minecraft/config" and edit "microblocks.cfg"
  • Insert it the following lines: "ArchitectFriend:smoothBlock":0-15 "ArchitectFriend:smoothBricks":0-15 "ArchitectFriend:smoothBrickies":0-15
  • Save "microblocks.cfg"
  • Restart Minecraft
